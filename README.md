# Usage

<b>Compile project:</b>

mvn install

<b>Launch Server:</b>

mvn exec:java -Dexec.mainClass="com.sanyabane.server.App"

<b>Launch Client:</b>

mvn exec:java -Dexec.mainClass="com.sanyabane.client.App" -Dexec.args="localhost"

<i>First argument is server IP</i>
